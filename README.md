# Neverland - Proyecto Intermodular 1º DAW

Este proyecto consiste en el desarrollo de un portal web corporativo para una empresa ficticia llamada Neverland, un local de eventos infantiles especializado en la organizacion de cumpleaños, fiestas temáticas y actividades para niños.

El proyecto se realiza como parte del Proyecto Intermodular de 1º de Desarrollo de Aplicaciones Web (DAW), integrando los conocimientos adquiridos en los diferentes modulos del curso.

## Modulos incluidos

* Bases de Datos
* Lenguajes de Marcas
* Programación
* Sistemas Informáticos
* Entornos de Desarrollo
* MPO Ampliación de Programación
* Itinerario Personal para la Empleabilidad (IPE)

## Tema del proyecto

Neverland es un local de eventos infantiles que ofrece los siguientes servicios:

* Celebracion de cumpleaños
* Fiestas temáticas
* Animación infantil
* Alquiler de salas
* Servicios de catering

La web corporativa permitirá mostrar información sobre la empresa, sus servicios, precios y formas de contacto, simulando un entorno profesional real.

## Objetivo del proyecto

El objetivo es simular el desarrollo completo de una aplicación web profesional, incluyendo:

* Diseño de base de datos en MySQL
* Desarrollo web con HTML y CSS
* Aplicación en Java con JDBC
* Documentación tecnica del sistema
* Organizacion del proyecto mediante Git y GitHub

## Estructura del repositorio

Cada modulo del proyecto se encuentra organizado en su carpeta correspondiente.
