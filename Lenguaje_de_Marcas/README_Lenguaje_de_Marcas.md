# Neverland — Portal Web Corporativo

Portal web corporativo para **Neverland**, un local de eventos infantiles ficticio ubicado en Zaragoza, especializado en la organización de cumpleaños, fiestas temáticas y actividades para niños. Desarrollado con HTML5 y CSS3 como parte del Proyecto Intermodular de 1º DAW.

## Páginas

- **index.html** — Inicio: presentación del local, servicios y galería de instalaciones
- **tarifas.html** — Tarifas por turno (mañana, tarde y día completo)
- **reserva.html** — Formulario de solicitud de reserva
- **contacto.html** — Datos de contacto y mapa de ubicación

## Cómo visualizarla

No requiere instalación. Descarga o clona el repositorio y abre `index.html` con cualquier navegador moderno.

También se puede visualizar desde Visual Studio Code usando la extensión **Live Server**: abre el proyecto en VS Code, haz clic derecho sobre `index.html` y selecciona "Open with Live Server".

Las fuentes e iconos se cargan desde CDN, por lo que se necesita conexión a internet para verlos correctamente.
