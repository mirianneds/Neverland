# Análisis de datos - Neverland

Neverland es un local de eventos infantiles dedicado a la organización de cumpleaños, fiestas temáticas y actividades para niños.

La empresa permite a los clientes reservar eventos mediante un formulario de contacto disponible en la web corporativa.

Para gestionar el funcionamiento del negocio, se necesita una base de datos que permita almacenar la información relacionada con clientes, reservas, eventos y servicios.

## Información que gestiona la empresa

El sistema debe permitir almacenar:

- Datos de clientes
- Datos de empleados
- Tipos de servicios ofrecidos
- Eventos organizados
- Reservas realizadas
- Pagos de los eventos

## Entidades principales

- Cliente
- Empleado
- Evento
- Reserva
- Servicio
- Pago
